CS 3502 Project 3 - File Manager

OVERVIEW
This project is a graphical file manager built using Python and Tkinter.  
It demonstrates CRUD operations and interaction with the operating system file system.

FEATURES
- Create files and folders
- Read and display file contents
- Edit and save files
- Rename files and directories
- Delete files and empty directories
- Navigate directories
- Display file metadata (size and modification time)
- Error handling for invalid operations

REQUIREMENTS
- Python 3
- tkinter
